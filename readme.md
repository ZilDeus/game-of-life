# GOF

Simple game of life in opengl and c

### Dependencies

* mingw if you're on windows 10
* glfw3

### Running

```
$ git clone https://github.com/ZilDeus/game-of-life YOUR_DIR
$ cd YOUR_DIR
```
* windows
```
$ generate.bat
$ cd winBuild\
$ make run
```
* linux
```
$ generate.zsh
$ cd unixBuild\
$ make run
```
